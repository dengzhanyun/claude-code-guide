Claude Code + DeepSeek V4 Pro

一条龙安装教程 · 装好就能用 AI 写代码

蚊子 · 小d

1

安装 VS Code

💻VS Code 官网下载code.visualstudio.com · 免费→
1
双击安装包，一路下一步
2
打开 VS Code，按 Ctrl+Shift+X 打开扩展商店
3
搜索 Claude Code，安装 Anthropic 官方插件
4
Ctrl+, 打开设置 → 搜索 claudeCode.disableLoginPrompt → 勾选
5
关掉 VS Code，等配好 CC Switch 再打开
2

安装 CC Switch

🔀CC Switch 下载GitHub · 免费开源→
1
双击安装包，一路下一步
2
打开 CC Switch，它会在右下角系统托盘常驻
CC Switch 安装后必须一直开着！右下角能看到图标就对了。关了就连不上大模型。
3

部署大模型

1
浏览器打开 platform.deepseek.com，注册登录
2
左侧菜单 → API Keys → 创建 → 复制 Key
sk-xxxxxxxxxxxxxxxxxxxxxxxx（只显示一次，马上复制保存）
3
打开 CC Switch → 点右上角 ＋ 新建配置
供应商名称DeepSeek
API Keysk-xxxxxxxx（刚才复制的）
请求地址https://api.deepseek.com/anthropic
最大强度思考开启 ✅
其他字段不用管。请求地址末尾不要加斜杠。
4
保存 → 回到主界面 → 点开关变绿
DeepSeek 新用户送免费额度，够用很久，用完了按量付费很便宜。
4

打开 Claude Code 开始用

1
打开 VS Code（确保 CC Switch 在系统托盘开着）
2
左侧栏点 AI 图标，打开 Claude Code 面板
3
输入 /model，看到 deepseek-v4-pro
4
输入 "你好" → AI 回复你了 → 大功告成！
🎉 安装成功！现在可以让 AI 帮你写代码了。
!
